# PulsatingPrism - 贸易模拟游戏

基于Unreal Engine 5开发的类Grand Ages Medieval贸易模拟游戏，展示了完整的游戏开发能力和系统设计能力。
但由于Github有上传空间限制，没有上传完整的项目，只上传了代码仓库

## 🎮 项目特色

- **贸易系统**: 自动贸易路线管理，实时市场价格波动
- **生产系统**: 26种商品的生产链，从原材料到成品
- **经济系统**: 税收、关税、人口消费模型
- **六边形网格**: 专业级策略游戏地图系统
- **模块化UI**: 可扩展的用户界面设计

## 🛠️ 技术实现

- **引擎**: Unreal Engine 5
- **编程语言**: C++
- **架构**: 模块化设计，数据驱动开发
- **核心系统**: 贸易管理、生产链、路径规划、经济模拟
